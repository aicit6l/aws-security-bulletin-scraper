# AWS Security Bulletin Scraper

⚠️ **Work in Progress** - Being refactored for AWS security focus

## What It Does
Automatically collects and tracks AWS security advisories and bulletins.

## Tech Stack
- Python 3.x
- Pandas (data manipulation)
- BeautifulSoup4 (web scraping)
- Requests (HTTP calls)
- LXML (HTML processing)

## Installation

```bash
git clone https://github.com/aicit6l/aws-security-bulletin-scraper.git
cd aws-security-bulletin-scraper
pip install -r requirements.txt
```

## Usage

```bash
python scraper.py
```
